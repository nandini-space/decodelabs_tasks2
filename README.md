# Titanic Survival Prediction Using AI

## Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning classification techniques.

## Technologies Used

* Python
* Google Colab
* Pandas
* Scikit-Learn
* Matplotlib

## Dataset

Titanic Dataset

## Machine Learning Algorithm

Random Forest Classifier

## Workflow

1. Loaded the Titanic dataset from a CSV file.
2. Performed data preprocessing and handled missing values.
3. Converted categorical data into numerical format.
4. Split the dataset into training and testing sets.
5. Trained a Random Forest Classifier.
6. Evaluated model performance using accuracy score.

## Visualizations

* Survival Distribution
* Feature Importance
* Confusion Matrix
* Accuracy Analysis

## Conclusion

The model successfully predicts passenger survival using demographic and travel-related features. This project demonstrates the fundamentals of supervised machine learning and classification.
